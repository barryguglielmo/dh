About this Site:

Hello World! This is a beta site for teaching and learning computational biology techniques. It is an experimental platform for me to get my teaching voice and start playing with concepts of what goes on behind the scenes of web based tools.

How it was built:

This site was constructed using Python, Django, JS, CSS, Bootstrap, Sqlite3, and deployed on a free Heroku Server. I plan to move the site to DigitalOcean soon. For tutorials on getting started building a website with this architecture follow THIS LINK. This is a tutorial series to build a blog and launch it on DigitalOcean. For information on launching a Django project on Heroku project follow THIS LINK.
