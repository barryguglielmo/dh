OPEN SOURCE PROGRAMMING





Open source programming is a collaborative effort to build free and accessible tools for making dope applicaitons or various types of software tools. From gaming to cancer research open source programming can help you build impressive applications.

This site will focus on the use of Python and R, but I will do my best to give advice and concepts that can be translated to any language. If you have never coded before I strongly suggest going to my Getting Started Tutorial. There you will find resources and various lesson plans that will help get you off the ground.

If you already have a computational language under your belt then it is time for you to start seeing some problems that you may encounter in the field. For this you can go to the Dojo to train some of your skills with real life problems.

If you want to keep up to date with what is going on in the world of computational biology you should check out The Buzz. It has articles and key concepts of the field.

I also have many of my Python and R scripts available for you to disect on my GitHub which you are free to access. I also have a section of this site dedicated to Python and R programming with references to some of my favorite modules and libraries that they provide. This section is called R-ython. Remember, programming is about being lazy, do not reinvent the wheel. Use all the tools available to you.
